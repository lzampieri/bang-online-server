# Bang online server
Server for bang online.
Made with `Node.js`

### URL
Run by default on `http://localhost:4001`
Port can be modify setting the PORT enviroment variable

### Install
To install, after cloning, `npm install`

### Run server
#### Dev mode
`npm run dev`

#### Hosting mode
`npm run start`

### Hosting
Hosted at `https://bang-online-server.herokuapp.com/`
